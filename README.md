# Weather App (Practice Project)

Basic full-stack app built for practicing MERN fundamentals.

### Tech Used
- React (Vite)
- Express.js
- Node.js
- MongoDB Atlas
- Render (Backend)
- Vercel (Frontend)

### Features
- Add / update weather for any city
- Search weather by city
- Fully deployed backend + frontend

### Live URL
Frontend: https://weather-app-react-express-mongodb.vercel.app/
Backend: https://weather-app-react-express-mongodb.onrender.com
